Securing the Cloud, Empowering Developers

Accomplished security leader with 10+ years of experience in cloud security, identity management, and technical advocacy. Currently serving as Head of Security Advocacy at Datadog, building research programs and fostering developer communities. Proven track record of architecting secure cloud solutions, leading engineering teams, and speaking at major industry conferences including BlackHat and AWS re:Invent.

Currently leading Datadog SecurityLabs at Datadog, I've built this brand from the ground up to reach over 80,000 unique readers per quarter. My work focuses on bridging the gap between security and development teams, making security practices more accessible and practical for modern cloud environments.

My technical expertise spans across cloud security architecture, identity management, DevSecOps, and security automation. I'm passionate about open source security tools and have contributed to Mozilla's security ecosystem, including the Enterprise Defense Platform and various AWS security utilities.
